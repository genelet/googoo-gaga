# Star Wars Prototypes (Googoo Gaga)

A collection of experimental Python scripts for Star Wars-themed game concepts, primarily using the `pygame` library.

## Project Structure

- **`web/`**: A modern React/TypeScript version of the game playable in any web browser.
- **`starwars_battlefront_2_3d.py`**: The original functional Python script (2D movement).
- **`starwars_battlefront_2.py`**: A conceptual design document for multiplayer features.
- **`starwars_fighting_game.py`**: character lists for 'rebels' and 'galactic_empire'.

## Requirements

### Python
- Python 3.x
- `pygame`

### Web
- Node.js (v18+)
- npm

## How to Run

### Web Prototype (Recommended)
1. Navigate to the web directory:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to the URL provided (usually `http://localhost:5173`).

### Python Prototype
```bash
python starwars_battlefront_2_3d.py
```
